# Módulo 06 - Aplicação com React Native

Neste módulo foi desenvolvido uma aplicação em React Native para listagem de usuários do Gitihub. Depois de adicionar o usuário, clicando sobre o item da lista, você pode verificar na tela de detalhes os repositórios que este usuário favoritou (marcou Star).

## O que foi utilizado?

- Axios (comunicação da API com o Github)
- Styled Components (estilização dos componentes)
- Reactotron (debbug da aplicação)
- React Navigation Stack (para navegação entre telas)
- Async Storage (para persistência dos dados)
- Configuração de ambiente com ESLint, Prettier & EditorConfig

## Features

- Adicionar usuário do Github
- Listar usuários inseridos
- Ver detalhes dos usuários
- Lista de repositórios com Star
